Pendahuluan#

Dataset yang digunakan dalam analisis ini adalah “Tic-Tac-Toe Endgame” yang bersumber dari UCI Machine Learning Repository. Dataset ini berisi semua kemungkinan konfigurasi papan permainan Tic-Tac-Toe di akhir permainan, di mana ‘x’ diasumsikan sebagai pemain pertama. Tujuan dari dataset ini adalah untuk keperluan klasifikasi biner, yaitu menentukan apakah suatu konfigurasi papan merupakan kemenangan untuk pemain ‘x’.

Dataset ini memiliki 958 data (instance) dan 10 atribut yang semuanya bersifat kategorikal. Dari 10 atribut tersebut, 9 atribut berfungsi sebagai fitur (merepresentasikan 9 kotak di papan) dan 1 atribut berfungsi sebagai kelas target (menunjukkan hasil akhir permainan).

Langkah 3: Pengecekan Missing Values#

Salah satu langkah paling krusial dalam persiapan data adalah memastikan tidak ada data yang hilang atau kosong (missing values). Data yang hilang dapat menyebabkan error saat pelatihan model atau menghasilkan prediksi yang tidak akurat.

Pada langkah ini, kita akan secara eksplisit menghitung nilai kosong di setiap kolom untuk mendapatkan konfirmasi akhir bahwa dataset kita sudah lengkap dan siap untuk tahap selanjutnya.

# df sudah ada dari langkah sebelumnya

print("--- Menghitung Jumlah Missing Values per Kolom ---")

missing_values = df.isnull().sum()

if missing_values.sum() == 0:

Langkah 4: Analisis Distribusi Nilai (Value Counts)#

Pada tahap ini, kita ingin mengetahui komposisi dari setiap kolom. Misalnya, pada kolom target class, ada berapa banyak hasil yang positive dan berapa yang negative? Pada kolom fitur top-left-square, berapa kali kotak tersebut diisi oleh x, o, atau b?

Untuk melakukan ini, kita akan menggunakan fungsi .value_counts() dari pandas pada setiap kolom. Ini akan menghitung frekuensi atau jumlah kemunculan setiap nilai unik dalam kolom tersebut.

print("--- Analisis Distribusi Nilai (Value Counts) per Kolom ---")

# Melakukan iterasi untuk setiap kolom di DataFrame
for column in df.columns:
    # Mencetak nama kolom untuk memberikan konteks
    print(f"\nDistribusi untuk kolom '{column}':")

    # Menggunakan .value_counts() untuk menghitung frekuensi setiap nilai unik dan menampilkannya
    print(df[column].value_counts())

Langkah 5: Visualisasi Distribusi Fitur#

Setelah kita melihat angka-angka distribusi pada langkah sebelumnya, akan jauh lebih mudah untuk memahami polanya jika kita mengubahnya menjadi grafik. Pada langkah ini, kita akan membuat diagram batang (bar chart) untuk setiap kolom guna memvisualisasikan frekuensi dari masing-masing kategori.

Visualisasi ini membantu kita untuk dengan cepat melihat:

Keseimbangan kelas pada kolom target (class). Distribusi isian (x, o, b) pada setiap posisi di papan permainan. Kita akan menggunakan library seaborn dan matplotlib untuk membuat semua plot ini secara otomatis dalam satu perulangan.

import matplotlib.pyplot as plt
import seaborn as sns

# Mengatur tema plot agar lebih menarik
sns.set_theme(style="whitegrid")

print("--- Membuat Visualisasi Distribusi untuk Setiap Kolom ---")

# Menyiapkan area plot dengan 5 baris dan 2 kolom
# figsize mengatur ukuran keseluruhan gambar
fig, axes = plt.subplots(nrows=5, ncols=2, figsize=(14, 22))

# Meratakan array 'axes' agar mudah di-loop
axes = axes.flatten()

# Melakukan perulangan untuk setiap kolom dan membuat countplot
for i, col in enumerate(df.columns):
    sns.countplot(x=df[col], ax=axes[i], palette='viridis', order = df[col].value_counts().index)
    axes[i].set_title(f'Distribusi Kolom: {col}', fontsize=14)
    axes[i].set_xlabel('')  # Menghilangkan label x agar tidak berantakan
    axes[i].set_ylabel('Jumlah')

# Menyembunyikan subplot yang tidak terpakai jika jumlah kolom ganjil
# (Untuk kasus ini, 10 plot pas, jadi tidak ada yang disembunyikan)
for j in range(len(df.columns), len(axes)):
    axes[j].axis('off')

# Merapikan tata letak agar judul dan label tidak saling tumpang tindih
plt.tight_layout()

# Menampilkan semua plot yang telah dibuat
plt.show()

Preprocessing (Encoding)#

Setelah kita memahami data kita melalui eksplorasi dan visualisasi, langkah selanjutnya adalah mempersiapkan data untuk model machine learning. Model machine learning tidak bisa memproses data teks (‘x’, ‘o’, ‘positive’, dll.), sehingga kita harus mengubahnya menjadi format angka. Proses ini disebut Encoding.

Untuk dataset ini, kita akan menggunakan dua jenis encoding:

  • Untuk Fitur (X): Kita akan menggunakan OneHotEncoder. Ini adalah pilihan terbaik karena fitur-fitur kita (posisi papan) bersifat nominal, artinya tidak ada urutan atau tingkatan (‘x’ tidak lebih baik dari ‘o’). OneHotEncoder akan membuat kolom-kolom biner baru untuk setiap kemungkinan nilai (misalnya top-left-square_x, top-left-square_o, top-left-square_b).

  • Untuk Target (y): Kita akan menggunakan LabelEncoder. Ini sudah cukup karena kita hanya perlu mengubah dua kategori (‘positive’ dan ‘negative’) menjadi nilai numerik seperti 1 dan 0.

# Import library untuk encoding
from sklearn.preprocessing import OneHotEncoder, LabelEncoder
import pandas as pd

# DataFrame 'df' sudah ada dari langkah-langkah sebelumnya

print("--- Memulai Proses Preprocessing Data (Encoding) ---")

# Memisahkan antara fitur (X) dan target (y)
X = df.drop('class', axis=1)
y = df['class']

# 1. Encoding Fitur (X) dengan OneHotEncoder
# sparse_output=False agar hasilnya berupa array numpy biasa, bukan sparse matrix
encoder_X = OneHotEncoder(sparse_output=False)
X_encoded_array = encoder_X.fit_transform(X)

# Mengubah hasil encoding menjadi DataFrame baru dengan nama kolom yang jelas
X_encoded_df = pd.DataFrame(X_encoded_array, columns=encoder_X.get_feature_names_out(X.columns))

print("\n--- Fitur (X) Setelah di-OneHotEncoder ---")
print(f"Bentuk data fitur baru (jumlah baris, jumlah kolom): {X_encoded_df.shape}")
print("5 baris pertama dari data fitur yang sudah di-encode:")
print(X_encoded_df.head())

# 2. Encoding Target (y) dengan LabelEncoder
encoder_y = LabelEncoder()
y_encoded = encoder_y.fit_transform(y)

print("\n--- Target (y) Setelah di-LabelEncoder ---")
# Menampilkan mapping dari teks ke angka ('negative' -> 0, 'positive' -> 1)
print(f"Mapping kelas target: {dict(zip(encoder_y.classes_, encoder_y.transform(encoder_y.classes_)))}")
print("Contoh 5 data target pertama yang sudah di-encode:")
print(y_encoded[:5])

Klasifikasi dengan Decision Tree#

Alur kerja lengkap untuk tahap klasifikasi:

  • **Membagi Data:**Kita akan membagi dataset menjadi dua: 80% untuk melatih model (data latih) dan 20% untuk menguji model (data uji).

  • Membuat & Melatih Model: Kita akan membuat model menggunakan algoritma Decision Tree Classifier, sama seperti pada contoh acuan Anda. Model ini akan “belajar” dari data latih.

  • Prediksi & Evaluasi: Setelah dilatih, model akan diminta untuk menebak hasil dari data uji. Kemudian, kita akan membandingkan tebakan model dengan jawaban sebenarnya untuk mengukur seberapa baik kinerjanya.

# Import library yang diperlukan untuk pemodelan dan evaluasi
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score, classification_report, confusion_matrix
import seaborn as sns
import matplotlib.pyplot as plt

# Variabel X_encoded_df dan y_encoded sudah ada dari langkah preprocessing sebelumnya

print("--- Memulai Tahap Klasifikasi ---")

# 1. Membagi Data Menjadi Data Latih dan Data Uji
# test_size=0.2 artinya 20% data digunakan untuk pengujian.
# stratify=y_encoded memastikan proporsi kelas 'positive' dan 'negative' seimbang.
X_train, X_test, y_train, y_test = train_test_split(
    X_encoded_df, y_encoded, test_size=0.2, random_state=42, stratify=y_encoded
)

print(f"\nData berhasil dibagi:")
print(f"Ukuran data latih: {X_train.shape[0]} baris")
print(f"Ukuran data uji: {X_test.shape[0]} baris")


# 2. Membuat dan Melatih Model Decision Tree
model = DecisionTreeClassifier(criterion='entropy', random_state=42)
model.fit(X_train, y_train)

print("\nModel Decision Tree berhasil dilatih.")


# 3. Melakukan Prediksi dan Evaluasi Model
print("\n--- Hasil Evaluasi Kinerja Model ---")
# Melakukan prediksi pada data uji
y_pred = model.predict(X_test)

# Menghitung akurasi
accuracy = accuracy_score(y_test, y_pred)
print(f"\nAkurasi Model: {accuracy * 100:.2f}%\n")

# Menampilkan Laporan Klasifikasi
# Kita menggunakan variabel encoder_y dari langkah preprocessing untuk mendapatkan nama kelas asli
class_names = encoder_y.classes_
print("Laporan Klasifikasi:")
print(classification_report(y_test, y_pred, target_names=class_names))


# 4. Menampilkan Confusion Matrix
# Variabel encoder_y juga digunakan di sini untuk label pada plot
cm = confusion_matrix(y_test, y_pred)
plt.figure(figsize=(8, 6))
sns.heatmap(cm, annot=True, fmt='d', cmap='Blues',
            xticklabels=class_names, yticklabels=class_names)
plt.title('Confusion Matrix', fontsize=14)
plt.xlabel('Prediksi Model', fontsize=12)
plt.ylabel('Label Sebenarnya', fontsize=12)
plt.show()
